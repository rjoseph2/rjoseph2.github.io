DOM (Document Object Model ) The DOM is a W3C (World Wide Web Consortium) standard. The Document Object Model is a cross platform and language independent appliction programing interface. DOM treats a ducuments (HTML, XML or XHTL) as a tree structure. Each part of a document (which is called as a node) is an object of DOM. This objects can be manipulated programatically. and the visible changes will be reflected in the document view. An XML document sample <student> <name> <firstName>FirstName</firstName> <lastName>LastName</lastName> </name> <phone> <mobile></mobile> </phone> <dob> <day>22</day> <month>09</month> <year>1988</year> </dob <country>India<country> </student> Dom represenation of above xml Above sample "student" is the root node for the document, root node has four child nodes (ie country,phone, dob and name). again the nodes phone, dob and name has child nodes The Html DOM Html DOM define the standard for document object model for html. Html DOM defines how to create, update and delete html document. It defines :- Html element Properties for html elements Events for all html elements Methods for accessing and modifying Html elements
